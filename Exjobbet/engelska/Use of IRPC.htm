How to use IRPC

Doubleclick on Command Prompt in the Source folder, resulting in the following:



Run IRPC with the following command:
%cc% MyProg.a68  <  Ex01.txt  >  Ex01_output



"Ex01.txt" contains the input data and the output ends up in "Ex01_output.txt".

Output data from the original program in year 1967 are listed on pages 39 to 54 in Exjobbet.
The examples tested in 2019 give exactly the same results !!

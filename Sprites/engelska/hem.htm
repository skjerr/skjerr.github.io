Summary

Fox = Animation tool FOr Xwindow

Sprite animation tool based on C++ and Xlib.
Portable design, proven in Apollo workstation, DEC Ultrix, DEC Alpha, Metawindow and Ubunu.
This version is tested on Ubuntu 20.04.2 running in Windows 10 WSL.
Many demo examples included.
Chessboard demo allowing two (human) players to play over internet.

Download Fox

Source code for the demos, static library containing the tool, and executables  for Ubuntu 20.04.2

History

In1991 I took a course in C++ and object oriented programming.
This animation tool was  my project work for this course.
The development was done in an Apollo workstation.

Since then I have ported it to DEC Ultrix, DEC Alpha, Metawindow and Ubunu.

The Ubuntu version, available here, was compiled with GCC g++.
